# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-24

### Added

#### Core Features
- Next.js 15 with App Router and TypeScript
- TinaCMS integration for content management
- Tailwind CSS 4 with CSS custom properties theming
- Dynamic theme system with ThemeProvider component

#### Layout System
- 3 Navbar variants: Floating, Fixed, Transparent
- 3 Footer variants: Full, Minimal, Centered
- 3 Homepage layouts: Standard, Hero-Full, Minimal
- Mobile-responsive navigation with slide-out menu
- PageLayout component for consistent page structure

#### Block Components (15 total)
- TextBlock - Simple title and paragraph content
- HeroBanner - Full-width hero with background image
- ImageTextLeft - Image left, text right layout
- ImageTextRight - Text left, image right layout
- CardsGrid - Grid of cards with icons (2-4 columns)
- CtaBox - Call-to-action box with buttons
- Faq - Expandable accordion Q&A
- ImageGallery - Photo grid with captions
- ContactInfo - Contact details with optional map
- Stats - Numbers/statistics display with icons
- Testimonials - Customer quotes (grid or carousel)
- Video - YouTube/Vimeo embed with auto-detection
- Timeline - Chronological event display
- Team - Team member grid with bios
- Divider - Visual separator with variants

#### Demo Content
- "Horizon Community Foundation" demo site
- Complete page content for: About, Events, Gallery, Contact, Get Involved
- 4 program pages: Youth Leadership, Adult Education, Community Wellness, Digital Skills
- Homepage with hero carousel and featured content

#### Documentation
- Getting Started guide
- Customization guide
- Block Types reference
- Deployment guide

### Technical Details
- React 19 with Server Components
- Git-based content storage (JSON files)
- Static site generation with ISR support
- 6 pre-configured font pairings
- Lucide React icons integration
- Full TypeScript type safety
